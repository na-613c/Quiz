## Описание

Викторина о животных с тремя уровнями сложности из 10 вопросов. Все рекорды хранятся в firebase.

## Использованные технологии 

- Retrofit2
- RxJava.
- Firebase.
- Comparator.
- Glide.
- Fragment
- ViewPager.
- RecyclerView.
